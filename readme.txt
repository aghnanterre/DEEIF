Projet DEEIF : dictionnaire des expressions d'intensité
 en français
 
 mai 2019
 
 Il s'agit d'un projet Django visant, dans un premier temps, à afficher
  sur une page web  les expressions de l'intensité de corpus rassemblés
  par Malgorztata, Ewa, et Clara.
  
  Il s'agirait d'afficher les expressions à partir de fichiers
  texte taggés au format xml, situés dans le répertoire corpus.
